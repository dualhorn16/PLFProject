usage:  reducer.py
        reducer.py source_file
        reducer.py source_file output_tex_file

to use the LaTeX file:
        0. install LaTeX
        1. put the included qtree.sty file into your /texmf folder
        2. $ latex [file_name]
        3. $ open [file_name].pdf
            or
           $ open [file_name].dvi
